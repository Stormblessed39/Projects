How to play the game


Selecting a tile

To select a tile to match, simply click on the tile. The tile will highlight blue if it is open. After clicking a tile the player may click another tile that they think to match with. If the tiles do not match, or the second tile is not open, then both tiles will revert back to their original color.

The menu

Game

Play -- if already playing, start a new game

Restart -- restart the current game

Numbered -- play a game based off of a game number

Toggle History -- toggle the history view on or off

Tournament -- start a game in tournament mode

Exit -- end the game

Sound

On -- play the tiles removed and the fireworks sounds

Off -- do not play the tiles removed or the fireworks sounds

Move

Undo -- undo the last move

Redo -- undo the undo
